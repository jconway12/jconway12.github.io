BeachBnb

A single page rental booking cite inspired by Airbnb, which uses a Ruby on Rails back end and a React-Redux frontend. Manages listing and reservation CRUD through custom transactional database operations using Active Record and PostgreSQL, allowing users to make reservations, register listings with photos, and write reviews. The back end is connected to AWS S3 for photo storage and organization, maintaining content security with AWS IAM, increasing scalability.

City Comparator

Pure JavaScript and D3 data visualization of Teleport API data. Users can filter quality of life ratings of urban areas through 16 different categories in a variety of ways through the search bar and side panel. If a user wishes to see the data for a specific city, simply enter that city into the search bar and click 'filter by city'. Otherwise, clicking 'all cities' allows the user to view all the available urban spaces in the API data. Data can also be filtered by minimum ratings and overall rating inclusion. API calls were reduced to one to increase filtering speed.

International Trade Forecast

Comprehensive forecast of variables in the US international sector in R. Deployed ARIMA, VAR, and exponential smoothing models in R to forecast the sector over two years. Incorporated macroeconomic theory to construct a thorough analysis of the results.
